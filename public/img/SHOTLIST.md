# CitiGrove — photography drop-in slots

The homepage is **photography-led** (Ffern register). Every image below is a
plain file in `public/img/`. To use your own photo, **replace the file with the
same name** — no code changes. JPG, sRGB, quality ~80, long edge ≥1600px.
Until a real product photo exists, food slots are seeded with on-brand
marketing imagery and the four ecommerce categories fall back to warm tonal
placeholders (intentional — they're waiting on your shots).

**Grade note:** keep everything in the warm cream/terracotta family (slightly
desaturated, gentle film contrast) so the page reads as one shoot, the way
Ffern does. Avoid cool/blue white-balance and hard studio flash.
